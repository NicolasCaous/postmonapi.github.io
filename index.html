Postmon

The Mongo Postman API

O que é?

Uma API para consultar CEP's de maneira fácil, rápida e gratuita.

Como utilizar?

Como é uma API baseada em REST, basta fazer a seguinte chamada:

http://api.postmon.com.br/cep/(CEP)

Onde (CEP) é o número do CEP que deseja consultar as informações. O resultado é retornado em formato JSON
